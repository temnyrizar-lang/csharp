## Теоретические вопросы
1. Для хранения числа 3,14 нужно использовать переменную типа double, так как это вещественное число.
2. Переменная типа string может хранить числа, если они заключены в двойные кавычки.
3. Вывод текста в консоль без перехода на новую строку осуществляется с помощью команды Console.Write.
4. На новую строку переходят с помощью \n (обязательно заключённое в двойные кавычки).
## Практика
Задание 1:

```csharp
using System;
class Program
{
    static void Main()
    {
        Console.WriteLine("Привет");
        Console.WriteLine("Привет");
        Console.WriteLine("Привет");
    }
}
```

```csharp
using System;
class Program
{
   ```

```csharp
        int age = 32; // здесь показан пример
```
